BOT DE OFERTAS TELEGRAM - Rodando 24h Online

PASSOS:
1. Crie um bot no Telegram via @BotFather e copie o TOKEN.
2. Edite bot_ofertas.py e coloque:
   TOKEN = "seu_token_aqui"
   CHAT_ID = "seu_chat_id_aqui"

3. Faça upload da pasta no Render.com:
   - Conecte ao seu GitHub ou envie ZIP
   - Runtime: Python 3
   - Build Command: pip install -r requirements.txt
   - Start Command: python bot_ofertas.py

4. Pronto! Seu bot enviará ofertas todos os dias às 09:00 automaticamente.
